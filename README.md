# ThreadPool_withCpp

使用C++实现一个线程池项目。
## 1固定式线程池
固定大小的线程池，在创建时就指定线程池中线程的数量。每当有任务提交到线程池时，线程池会启动一个线程来执行任务，直到达到线程池的最大线程数。

## 2缓存式线程池
动态调整线程数量，当有新任务提交时，如果线程池中有空闲线程，则会立即使用空闲线程执行任务；如果没有2空闲线程，则会创建一个新线程来执行任务。当线程空闲一段时间后，超过一定时间后会被回收。

## 3工作窃取线程池
当某个线程执行完自己队列的任务后，会从其他线程的队列中窃取任务来执行。这种设计可以充分发挥多核处理器的并行处理能力，提高整体的任务处理效率。

## 4计划线程池
在指定的时间间隔或时间点异步的执行任务。
